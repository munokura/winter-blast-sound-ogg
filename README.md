<h2>▼ファイル名と曲名</h2>
<p>基本的にフォルダで分けてありますが、ファイル名の後ろに'_l'がついているものはループ処理されています。
'_l_s'がついているものは、ショートバージョンでループしています。
一部ループさせるのが困難な曲があり、それは原曲のままoggにコンバートしています。</p>
